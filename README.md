# Binance_5Pairs
The Executable Binance Arbitrage System


## Getting Started
To install
```javascript
  pip install python-binance
```

to enable API requests from Binance using python

## Execution
```javascript
  python3 main.py
```

## Continuously Running in Background
```javascript
  nohup python3 main.py &
```
